# Power-BI-Dashboard-5
HR Consultancy Analytics Dashboard
Description / Purpose:
This Power BI dashboard provides a comprehensive analysis of HR consultancy data, enabling stakeholders to monitor employee trends, hiring performance, and operational efficiency. The goal is to transform raw HR data into actionable insights for better decision-making.

🛠️ Technologies Used:
Power BI – Data visualization & dashboard creation
Power Query – Data transformation & cleaning
DAX (Data Analysis Expressions) – Calculated measures and KPIs
Excel / CSV – Data source handling with DAX Query

✨ Features & Highlights:
Interactive dashboard with slicers and filters
Real-time KPI tracking
Drill-down capability for detailed insights
Clean and user-friendly UI design
Dynamic charts updating based on our selection.

📈 Key Visuals & Design Approach
1. Employee Distribution Analysis
Chart: Bar Chart / Column Chart: To visualize employee count across departments or roles which Helps identify workforce distribution and imbalance
2. Attrition & Retention Insights
Chart: Pie Chart / Donut Chart: To show employee attrition rate to Quickly highlights retention issues
3. Salary & Performance Trends
Chart: Line Chart is used To analyze trends over time which Helps in understanding growth and compensation patterns
4. KPI Cards
Visual Used: Card Visuals: Display key metrics like total employees, attrition rate and average salary.
5. Filters & Slicers: Enable dynamic exploration of data
Why: Improves user interactivity and analysis depth
💡 Business Impact / Insights:
1. Identified departments with high attrition rates
2. Helped in understanding salary distribution across roles
3. Enabled data-driven hiring strategies
4. Improved decision-making with real-time insights and Highlighted performance trends for workforce optimization.
